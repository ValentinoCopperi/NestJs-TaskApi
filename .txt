
Este proyecto es una red social de tareas que permite a los usuarios crear posts, comentar y dar likes a las publicaciones de otros, así como gestionar sus tareas con distintos estados. La API está desarrollada en NestJS utilizando TypeScript y se conecta a una base de datos MongoDB. Para la autenticación y la seguridad, implementa guards y sesiones con JWT. Además, utiliza WebSockets para la comunicación en tiempo real, asegurando que las interacciones, como nuevos comentarios y cambios en los estados de las tareas, se reflejen de forma instantánea para todos los usuarios conectados.

This project is a task-based social network that allows users to create posts, comment, and like others' posts, as well as manage their tasks with different statuses. The API is developed using NestJS with TypeScript and connects to a MongoDB database. For authentication and security, it implements guards and JWT-based sessions. Additionally, the project utilizes WebSockets for real-time communication, ensuring that interactions such as new comments and task status updates are instantly reflected for all connected users.
